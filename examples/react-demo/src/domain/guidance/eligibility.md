# Eligibility

The Neighborhood Futures Fund awards micro-grants to organizations
serving the fund's service area. This page covers who can apply.

## Who can apply

Registered nonprofits, schools, and local government offices can apply
with no minimum time in operation. Informal community groups (not
separately incorporated) can also apply, but must have been active for
at least six months, since the fund needs some evidence of an ongoing
track record before funding an unincorporated group.

## Service area

The fund only accepts applications from organizations whose project
serves one of four areas: Riverside, Downtown, Eastside, or the whole
county. Choose "Countywide" only if the project genuinely serves
multiple areas, not as a default when unsure; an application that picks
Countywide without a multi-area project gets sent back for clarification
before review.

## One application per cycle

An organization may submit only one application per funding cycle,
regardless of how many projects it runs. If an organization has
multiple project ideas, it should apply with the one it wants funded
most and can apply again next cycle with another.
